Users Table

Method Path Description
GET /user Get all users
GET /user/:id Gets single user by userID
POST /user/add Adds a one user
PATCH/PUT /user/update/:id Edits user details by userID
DELETE /user/delete/:id Delete user by userID

Products Table

Method Path Description
GET /product Get all items
GET /product/:id Gets single item by itemID
POST /product/add Adds a single item
PATCH/PUT /product/update/:id Edits a single item details by itemID
DELETE /product/delete/:id Delete item by itemID
