Users Table

Method Path Description
GET /users Get all users
GET /users/:id Gets single user by userID
POST /users/add Adds a one user
PATCH/PUT /users/update/:id Edits user details by userID
DELETE /users/delete/:id Delete user by userID

Items Table

Method Path Description
GET /items Get all items
GET /items/:id Gets single item by itemID
POST /items/add Adds a single item
PATCH/PUT /items/update/:id Edits a single item details by itemID
DELETE /items/delete/:id Delete item by itemID
